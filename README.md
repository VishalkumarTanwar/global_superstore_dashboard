
# Global Super Store Sales Dashboard

This project presents an interactive business intelligence dashboard built using Power BI to analyze the performance of a global retail superstore. The dashboard provides insights into sales, profit, customer behavior, and regional performance to help support data-driven decision making.

 Project Overview

The Global Super Store Dashboard visualizes key business metrics such as:

Total Sales

Total Profit

Customer Count

Order Count

Market Performance

Monthly Sales Trends

It enables users to explore data dynamically using filters and interactive charts
